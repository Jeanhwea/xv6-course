XV6 是 MIT 开发的一个教学用的完整的类 Unix 操作系统，并且在 MIT 的操作系统课程
[[https://pdos.csail.mit.edu/6.828/2019/index.html][MIT 6.828]] 中使用。通过阅读并理解 XV6 的代码，可以清楚地了解操作系统中众多核心的
概念。 目前 XV6 操作系统支持 x86 和 riscv 两种体系结构，这里以 x86 版本为基础，
对其源代码进行讲解。
